# scroll_project
Front : React 기반, Back : spring boot를 이용하여 데이터를 받고 게시판 내용을 해당 리스트 출력을 해보고, 선택 후 이전화면으로 돌아갈 시 선택된 데이터에 포커싱을 맞춘다.


# 작업 환경
- React
- Spring boot
- Mysql
- nodeJS 16.10.8
- My-Batis
